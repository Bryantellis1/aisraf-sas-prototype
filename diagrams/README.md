## Purpose

Governed diagram source, render, thumbnail, accessibility, and registry folder for the AISRAF v0.1.2 public evaluation package.

## What Belongs Here

Build Package 13 adds the first public-quality visual pack for AISRAF v0.1.2:

- [diagram-registry.yaml](diagram-registry.yaml) - governed registry with purpose, stakeholder, viewpoint, target docs, alt text, status, and review notes.
- [release-v0.1.2/README.md](release-v0.1.2/README.md) - visual pack index.
- [release-v0.1.2/png/](release-v0.1.2/png/) - accepted full-size PNG diagrams.
- [release-v0.1.2/thumbnails/](release-v0.1.2/thumbnails/) - generated PNG thumbnails.
- [release-v0.1.2/source/](release-v0.1.2/source/) - source provenance notes.
- [release-v0.1.2/svg/](release-v0.1.2/svg/) - SVG status notes.

## What Does Not Belong Here

Release binaries, DOCX/PDF/PPTX/ZIP exports, runtime proof, cloud proof, database proof, Terraform proof, event-bus or telemetry proof, AM4 adapter execution proof, AL5 diagrams, stale diagrams, unreviewed source visuals, or diagrams that introduce marketplace, production-readiness, external post-back, or unsupported current-AM4 claims.
