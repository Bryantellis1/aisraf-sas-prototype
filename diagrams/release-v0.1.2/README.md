# AISRAF v0.1.2 Release Visual Pack

Work package: WP-13-FIRST-PUBLIC-VISUAL-PACK-AND-PUBLICATION-EXPORT

This folder contains the first public-quality AISRAF v0.1.2 visual pack accepted for governed Markdown documentation. The pack is public source-available evaluation-only release material. It is not open source, not production software, not marketplace publication proof, and not AM4 adapter execution proof.

## Contents

- `png/` - accepted full-size PNG diagrams for GitHub Markdown.
- `thumbnails/` - generated 480 px preview PNGs with matching file names.
- `source/` - source-note folder for provenance and future editable source handling.
- `svg/` - SVG-note folder; no SVG renders are included in this pack.

## Boundary

The visuals are documentation and governance assets. They do not add runtime behavior, external adapter execution, cloud deployment, database runtime, Terraform, event bus, telemetry, marketplace publication, production readiness, or AL5 closed-loop autonomy.

Publication export uses the same Markdown wording as the repository docs. DOCX and PDF outputs are deferred to GitHub Release assets or a later exact-path release authorization; no DOCX, PDF, PPTX, or ZIP file is committed by this pack.
